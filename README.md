# Backend Authentication Demo (bcrypt + JWT)

This project demonstrates a simple backend authentication system using:

- Node.js
- Express
- MySQL
- bcrypt (password hashing)
- JWT (authentication & authorization)

## Features
- User Registration
- Secure Password Hashing
- Login with JWT Token
- Protected Routes using Middleware
